# Movie Review Social App

* **Focus:**
  * Authentification with JWT
  * Apollo mutation with file upload
  * Apollo cache
  * User profile page
  * Explore page, Following page
  * 
* **Technologies used:** 
  * ReactJS
  * Redux
  * Formik
  * react-dropzone
  * styled-components
  * apollo-server-express
  * Mongoose
  * JWT
  * MongoDB
